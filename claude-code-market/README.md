# Claude Code Market

这个目录提供 Claude Code 可安装插件，包含以下三个 skill：

- `thread-coordination`
- `thread-goal-worker`
- `git-commit`

## 安装

在 Claude Code 里添加远程 marketplace：

```text
/plugin marketplace add Ghost233/ghost-agent-market --sparse claude-code-market
```

安装插件：

```text
/plugin install ghost-agent-workflow@ghost-agent-market
```
